# 📚 The Academic Sanctuary

Aplikasi manajemen akademik mahasiswa berbasis Flutter dengan database SQLite.

## Fitur
- Register & Login dengan validasi
- Dashboard dengan statistik SKS dan jadwal kuliah
- Tambah & edit jadwal kuliah
- Tab Schedule, News, dan Profile
- Data tersimpan permanen (SQLite)

## Teknologi
- Flutter & Dart
- sqflite (database lokal)
- google_fonts

## Cara Menjalankan
```bash
git clone https://github.com/AriaMahesa/the-academic-sanctuary.git
cd the-academic-sanctuary
flutter pub get
flutter run
```
